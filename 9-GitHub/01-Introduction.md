
# **Introduction to GitHub**

## üîπ What is GitHub?

**GitHub** is a cloud platform where developers store, manage, and track their code using **Git**.

Think of GitHub as:

* A **code hosting platform**
* A place to **collaborate** with others
* A platform for **version control**
* A **portfolio** for developers
* A place to store your **projects safely**

GitHub uses **Git** behind the scenes.
Git = version control tool
GitHub = website where you upload and manage Git repositories.

---

## Why GitHub?

GitHub makes it easy to:

* Store your code online
* Track changes in your code (version control)
* Work with teams on the same project
* Share your work publicly or privately
* Deploy projects on GitHub Pages, Vercel, etc.
* Showcase your skills to recruiters

## **Installing GitHub (Actually Git)**

To use GitHub, you must install **Git** on your computer.

---

### **1. Download Git**

Go to the official site:

üëâ [https://git-scm.com/downloads](https://git-scm.com/downloads)

Choose your OS:

* Windows
* macOS
* Linux (Ubuntu, Fedora, etc.)

---

### **2. Install Git on Windows**

Steps:

1. Download `Git for Windows`.
2. Run the installer.
3. Keep all default settings.
4. Click **Next ‚Üí Next ‚Üí Install**.
5. After installation, open **Command Prompt** or **Git Bash**.

Check installation:

```
git --version
```

If it shows a version, Git is installed successfully.

---

### **3. Install Git on macOS**

Use Homebrew:

```
brew install git
```

Or download from Git website.

Check Git version:

```
git --version
```

---

### **4. Install Git on Linux (Ubuntu)**

Open Terminal:

```
sudo apt update
sudo apt install git
git --version
```

---

### **5. Configure Git (Important!)**

Once Git is installed, configure your GitHub username & email:

```
git config --global user.name "Your Name"
git config --global user.email "yourgithubemail@example.com"
```

Check if saved:

```
git config --list
```

---

### **6. Create a GitHub Account**

Go to:

üëâ [https://github.com](https://github.com)

* Sign up
* Choose a username
* Confirm email

Your GitHub profile is now ready.
